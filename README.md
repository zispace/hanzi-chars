### 文件格式

数据以纯文本方式展示，`file.txt`文件格式（`UTF-8`格式编码）如下：

```plaintext
# 注释：字表名
# 注释：汉字数
# 注释：其他说明
 # 空行分隔
汉 # 每字一行（或一组字，以括号表示繁体、异体字、旧字形等）
字
简〔繁〕
简〔异〕〔〖IDS〗〕

```

如需读取字表，通常仅需忽略#开头的行和空行，读取每行第一个字符。

一般地，如下用括号符号区分不同字形：
- 【】表示新字形或简化字；
- ［］、〔〕或﹝﹞表示繁体字（传承字、正字）、异体字以及旧字体等，有时不区分繁体异体；
- 〖〗表示IDS，用于表示缺少编码或无法显示的汉字，整体视作一个字。
- ※表示特殊说明；
- ◎表示占位字符，通常配合〖〗使用。
