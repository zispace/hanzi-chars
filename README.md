# 常见汉字字表和字符集

![](https://img.shields.io/badge/%E6%B1%89%E5%AD%97_hanzi-005AF0?style=for-the-badge)

本仓库仅梳理常见的汉字字表和字符集（汉字来自CJKV）

## 说明

数据以纯文本方式展示，`file.txt`文件格式（`UTF-8`格式编码）如下：

```plaintext
# 注释：字表名
# 注释：汉字数
# 注释：其他说明
 # 空行分隔
汉 # 每字一行（或一组字，以括号表示繁体、异体字、旧字形等）
字
简〔繁〕

```
